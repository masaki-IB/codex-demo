# 未来創造株式会社 コーポレートサイト

名刺代わりに利用できる1ページ構成のランディングページと、会社情報・プライバシーポリシーの個別ページです。シンプルなHTML/CSSのみで構築しているため、軽量でメンテナンスもしやすい構成になっています。

## 構成
- `index.html`: トップページ。事業紹介・実績・会社情報ダイジェスト・お問い合わせ導線を掲載。
- `company.html`: 詳細な会社情報や沿革・アクセス情報を掲載。
- `privacy.html`: 個人情報保護方針。
- `styles.css`: 3ページ共通のスタイル。レスポンシブ対応＆高コントラスト配色。
- `assets/`: ヒーローイラストや実績モック、OGP用画像を配置。

## ローカルプレビュー
ブラウザで `index.html` を直接開くか、簡易サーバーで配信するとリンク遷移時も崩れません。

```bash
# 任意のポートでプレビュー
python3 -m http.server 4173
# または
npx serve .
```

## GitHubで公開する手順（GitHub Pages）
1. GitHubで新しいリポジトリを作成し、`main` ブランチにこのフォルダの内容をpushします。
2. リポジトリの Settings → Pages を開き、Source を `Deploy from a branch`、Branch を `main / root` に設定して保存します。
3. 数分で `https://<ユーザー名>.github.io/<リポジトリ名>/` が公開されます。SNS用画像などのパスは相対指定なので、追加設定は不要です。
4. コンテンツを編集したら `git add` → `git commit` → `git push` で更新され、Pagesも自動的に反映されます。

## カスタマイズのヒント
- `assets` 配下のSVGや `og-image.png` を自社ビジュアルに差し替えるとブランド感が高まります。
- お問い合わせボタンのURL（Formspree等）を本番用に差し替えてください。
- 色やフォントは `styles.css` のCSSカスタムプロパティ（`--color-*`）を書き換えると一括変更できます。
